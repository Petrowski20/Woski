# Woski

> **Compete with your friends. Not against the house.**

Woski is a prediction platform for sports and esports competitions.

Create private leagues, make predictions, compete with friends and follow your favourite tournaments — all without betting or real money.

The project started as a private prediction app for the 2026 FIFA World Cup and is now evolving into a generic platform capable of supporting multiple games, competitions and customizable scoring systems.

---

## Vision

Woski is built around one simple idea:

> **Competition should be social, not gambling.**

There are no bets.
There is no real money.
There are no gambling mechanics.

Instead, Woski focuses on:

- 🎯 Friendly competition
- 🏆 Private leagues
- 📊 Rankings and statistics
- ⚙️ Flexible scoring rules
- 🌍 Multiple sports and esports

Whether it's a World Cup, Worlds, MSI or a friendly competition between friends, Woski aims to bring everyone together in one place.

---

## Planned Competitions

The platform is being designed to support multiple games.

Initial targets include:

- ⚽ Football (National Teams)
  - FIFA World Cup
  - UEFA Euro

- 🎮 League of Legends
  - Worlds
  - MSI

- 🎯 VALORANT

The architecture is intentionally generic so more competitions can be added in the future without redesigning the application.

## Development

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The application will be available at:

```
http://localhost:3000
```
